Now we need 3 functions. One to generate symmetric matrices A which is included in myRfunctions.R, one to create time seires data for which there are two options in myRfunctions.R, and one to calculate \(P(X|A)=\dfrac{(detA)^{N/2}}{(2\pi)^{NS/2}} exp(-\dfrac{1}{2}\sum_{a=1}^{N}\sum_{ij}^S x_i^a A_{ij} x_j^a)\). We compute \(\sum_{a=1}^{N}\sum_{ij}^S x_i^a A_{ij} x_j^a\) by finding \(\sum_{i=j}X^TAX\)

P_x<-function(X, A)
{
    # note: we're always returning the log of P(X|A)
    
    S=nrow(A)
    N=ncol(X)
    
    #determinant returns a list of the absolute value of the determinat(modulus) and the        sign
    logDA = determinant(A, logarithm = TRUE)
    logDA = logDA$modulus * logDA$sign
    
    out <- N/2 * (logDA - S * log(2 * pi)) - 1/2 * sum(diag(t(X)%*%A%*%X))
    
    return(as.numeric(out))
}

Monte Carlo Integral

Given an SxN matrix of times series data X, our goal is to find the average probability, \(P(X|\mu,\sigma,d)\). We use a function created to compute the Monte Carlo Integral, B times for 1 set of \(\mu,\sigma,d\)

Then we optimize this for one set of \(\mu, \sigma, d\) to see if it can infer parameters close to the actual parameters which were used to make the data

optim(par=c(mu,sigma,d),fn=MonteCarlo,X=X,B=B, method = 'L-BFGS-B', 
      lower = c(-10, 0, 0), upper = c(10, 10, 100),
      control = list(fnscale = -1))
